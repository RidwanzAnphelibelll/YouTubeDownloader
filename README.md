# YouTube Downloader.

```Alat untuk mengunduh mengunduh video dan audio dari YouTube.```

## Instalasi

### 1. Download Termux
```bash
https://f-droid.org/packages/com.termux/
```

### 2. Update dan Upgrade
```bash
apt update && apt upgrade -y
```

### 3. Install Python
```bash
apt install python3 -y
```

### 3. Install Git
```bash
apt install git -y
```

### 4. Setup Penyimpanan
```bash
termux-setup-storage
```

### 5. Clone Repository
```bash
git clone https://github.com/RidwanzAnphelibelll/YouTubeDownloader 
cd YouTubeDownloader
```

### 6. Jalankan Program
```bash
python3 Run.py
```
